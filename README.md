# the-lujain-quiz

A sleek, interactive 19-question web quiz designed to test how well people know me. It features smooth scroll-driven effects, a light/dark theme toggle, custom photo slots, and a real-time global leaderboard.

✨ **[Live Demo Available Here](https://lujainiscool.github.io/the-lujain-quiz/)**

---

### 🚀 Features
* **Interactive Quiz:** 19 scrambled multi-choice questions with instant feedback.
* **Global Leaderboard:** Powered by **Supabase** to securely store and fetch high scores from every player across devices.
* **Modern UI:** Built with custom CSS variables, parallax image scroll effects, and native Intersection Observers for immersive transitions.
* **Theme Support:** Clean toggling between dark and light modes using localized browser storage.

---

### 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend Database:** Supabase (PostgreSQL)
* **Hosting:** GitHub Pages
